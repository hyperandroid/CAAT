Class Detail
CAAT.Button()
This class works as a UI Button.

To fully define the button, four images should be supplied as well as a callback function. The images define different button states:

  • Normal state
  • Pointed
  • Pressed
  • Disabled

It is only compulsory to supply an image for the normal state. All images must be supplied in a single image strip which containes all button states, concretely in a CAAT.CompoundImage instance.
