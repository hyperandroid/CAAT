Class Detail
CAAT.ImageProcessor()
ImageProcessor is a class to encapsulate image processing operations. These image processing manipulates individual image pixels and from an array of pixels builds an image which can be used as a pattern or image.

This class pre-creates a canvas of the given dimensions and extracts an imageData object to hold the pixel manipulation.
