CAAT.ScaleBehavior()
ScaleBehavior applies scale affine transforms in both axis. StartScale and EndScale must be supplied for each axis. This method takes care of a FF bug in which if a Scale is set to 0, the animation will fail playing. This behavior specifies anchors in values ranges 0..1
